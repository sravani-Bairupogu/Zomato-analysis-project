# Zomato-analysis-project
## Description
This project involves an analysis of the Zomato dataset to uncover insights related to restaurant trends and customer preferences.
Technologies Used
SQL: Used for data extraction and manipulation from the Zomato database.
Excel: Employed for data cleaning and initial analysis to organize and summarize key metrics.
Tableau: Utilized for creating interactive visualizations to explore trends and insights.
Power BI: Leveraged for developing comprehensive dashboards to present findings to stakeholders.
